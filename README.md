# تطبيق المهام الدينية

هذا التطبيق يساعدك على متابعة الصلوات الخمس والورد اليومي من الصلاة على النبي (200 مرة مع كل صلاة).

## كيفية التشغيل

1. تأكد من تثبيت Flutter SDK على جهازك.
2. افتح المجلد `religious_tasks_app` في سطر الأوامر (Terminal).
3. حمل المكتبات اللازمة بالأمر:
   ```bash
   flutter pub get
   ```
4. شغل التطبيق على هاتفك أو المحاكي بالأمر:
   ```bash
   flutter run
   ```

## المميزات
- قائمة بجميع الصلوات.
- خانة لوضع علامة "صح" عند الانتهاء.
- واجهة بسيطة وواضحة باللغة العربية.
