# تطبيق المهام الدينية (Religious Tasks App)

تطبيق إسلامي شامل يساعدك على متابعة عباداتك اليومية وتنظيم وقتك مع ميزات متقدمة.

## الإصدار الحالي
**v3.5.2+14**

## المميزات الجديدة والتحسينات
- **نظام الأذان المطور:** دعم تشغيل الأذان في الخلفية باستخدام خدمات أندرويد الأصلية لضمان الدقة.
- **تفاصيل الأذكار:** شاشة مخصصة لعرض الأذكار مع عدادات وتنبيهات.
- **خدمة الراديو:** إمكانية الاستماع إلى إذاعات القرآن الكريم مباشرة.
- **التنبيهات الذكية:** نظام إشعارات متطور لتذكيرك بالأذكار والمهام الدينية في وقتها.
- **إدارة الصلاحيات:** شاشة "Onboarding" جديدة لتسهيل إعطاء الصلاحيات اللازمة للتطبيق (الموقع، الإشعارات، العمل في الخلفية).
- **تحديثات تلقائية:** ميزة تحديث الأذكار والبيانات في الخلفية لضمان عمل التطبيق بكفاءة.

## كيفية التشغيل (للمطورين)
1. تأكد من تثبيت Flutter SDK.
2. تنفيذ الأمر `flutter pub get` لتحميل المكتبات.
3. تشغيل التطبيق باستخدام `flutter run`.

## لبناء نسخة APK لصديقك
قم بتنفيذ الأمر التالي في Terminal:
```bash
flutter build apk --split-per-abi
```
ستجد الملفات الناتجة في المسار:
`build/app/outputs/flutter-apk/`
